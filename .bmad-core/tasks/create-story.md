# Task: Create User Story

**Owner**: PM (Petra)
**Trigger**: Stakeholder request or roadmap item

## Template

```markdown
## Story: <Title>

**As a** <role: marketing team member / platform admin / API consumer>
**I want to** <action>
**So that** <business value>

### Context
<Background. Why now. What changed.>

### Acceptance Criteria
- [ ] Given <precondition>, when <action>, then <outcome>
- [ ] Given <precondition>, when <action>, then <outcome>

### Out of Scope
- <What explicitly NOT included>

### Dependencies
- <Other stories or specs this depends on>

### Success Metric
<How we measure this is working in production>
```

## Story Types for This Project

| Type | Example |
|------|---------|
| Auth | As a user, I want to refresh my token without logging in again |
| Task | As a marketer, I want to submit a task and see AI progress in real-time |
| Knowledge | As a team owner, I want to upload brand guidelines for agents to use |
| Approval | As a manager, I want to review and approve AI-generated content |
| Monitoring | As a platform admin, I want to see token costs per project |
