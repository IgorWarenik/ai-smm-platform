# Kits

Kits define what the AI Marketing Platform must do.

## Conventions
- Start with `cavekit-overview.md`.
- Domain kits use R-numbered requirements and testable acceptance criteria.
- `[GAP]` criteria describe known mismatches between desired behavior, docs, and implementation.
- Keep implementation details in `context/plans/`, not in kits.
